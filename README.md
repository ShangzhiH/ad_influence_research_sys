# ad_influence_research_sys
## 介绍
实习的时候在GroupM广告公司做的一个做数据分析的web小应用，ui基于RShiny，图表基于RChart，模型是PCA+回归+时间序列。

## 目的
建模分析某品牌汽车过去一年的广告投放和销量的关系
